# J3K Documentario

Portal documental corporativo J3K listo para GitHub Pages.

## Archivo principal

El archivo principal es:

```text
index.html
```

## Cómo subir a GitHub

1. Descomprime este ZIP.
2. Sube el contenido descomprimido al repositorio.
3. Verifica que `index.html` quede en la raíz del repositorio.
4. En GitHub ve a:
   Settings / Ajustes → Pages
5. Selecciona:
   Deploy from branch → main → /root
6. Guarda los cambios.

GitHub Pages publicará automáticamente el portal usando `index.html`.
